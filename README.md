## For developers

Clone the source locally:

```sh
$ git clone https://github.com/selester12/PastelPastures.git
$ cd PastelPastures/pastel-pastures
```

Install project dependencies:

```sh
$ npm install
```
Start the app:

```sh
$ npm start
```
